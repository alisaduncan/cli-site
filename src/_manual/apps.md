Manage Okta apps

## Usage
```bash
okta apps [-hV] [--batch] [--verbose] [COMMAND]
```

## Options
*None*

## Shared Options
```
      --batch           Batch mode, will not prompt for user input
  -h, --help            Show this help message and exit.
  -V, --version         Print version information and exit.
      --verbose         Verbose logging
```
