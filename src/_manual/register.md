Sign up for a new Okta account

## Usage
```bash
okta register [-hV] [--batch] [--verbose] [--company=<company>]
              [--email=<email>] [--first-name=<firstName>] [--last-name=<lastName>]
```

## Options
```
      --company=<company>   Company / organization used when registering a new Okta account
      --email=<email>       Email used when registering a new Okta account
      --first-name=<firstName>
                            First name used when registering a new Okta account
      --last-name=<lastName>
                            Last name used when registering a new Okta account
```

## Shared Options
```
      --batch           Batch mode, will not prompt for user input
  -h, --help            Show this help message and exit.
  -V, --version         Print version information and exit.
      --verbose         Verbose logging
```
